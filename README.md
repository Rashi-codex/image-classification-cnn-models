# 🐶🐱 Dogs vs Cats Classification using CNN

This project uses a **Convolutional Neural Network (CNN)** to classify images as either a dog or a cat.

---

## 📌 Project Overview

The goal of this project is to build a deep learning model that can accurately distinguish between images of dogs and cats. It is a binary image classification problem.

---

## ⚙️ Technologies Used

* Python 🐍
* TensorFlow / Keras
* NumPy
* Matplotlib
* OpenCV (optional)

---

## 📊 Dataset

The dataset consists of labeled images of: training and testing data
Images are preprocessed and resized before training the model.

---

## 🧠 Machine Learning Model

* Model: Convolutional Neural Network (CNN)
* Framework: TensorFlow / Keras
* Layers:

  * Convolutional Layers
  * Max Pooling Layers
  * Fully Connected Layers
  * Dropout (for regularization)

---

## 📈 Workflow

1. Import dataset
2. Data preprocessing (resizing, normalization)
3. Data augmentation
4. Build CNN model
5. Train the model
6. Evaluate performance
7. Make predictions

---

## 📉 Evaluation Metrics

* Accuracy
* Loss (Training & Validation)
* Confusion Matrix

---

## 📌 Project Features

* Data augmentation for better generalization

* Beginner-friendly deep learning project
